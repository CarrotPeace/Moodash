Welcome to My App!

Key functionalities

Mood Recording and Quotes Pushing

The user will select their current emotion from a list of pre-set emotions in the app, then the app will use an API to access a database of quotes and select the appropriate quote based on the user's emotion. Also, the app will access the user's location for functionality.

Quotes Collection

The user will have the option to "like" the quote they receive in functionality 1, and the app will save it to their quotes collection. The user can view their saved quotes in a separate section.

Personal Data Analysis

This functionality allows users to choose a certain date period and view an analysis of their recorded emotions.

Map Showing Moods by Region

This functionality allows users to view a map showing the recorded emotions of app users in different regions. The app will access the recorded emotions of app users in functionality 1 and display the number of each emotion in a certain area using different icons.
